# Python Data Handling Practical

This project is created as part of a Python data handling practical.

## Task Overview
- Created a list of numbers and calculated mean, median, minimum, and maximum manually (without NumPy).
- Created a dictionary of students with marks and identified the top 3 scorers.
- Read numbers from a text file and calculated their sum using file handling.

## Files Included
- `task1_data_handling.py` – Main Python script containing all logic
- `numbers.txt` – Text file containing numbers for file handling
